##Pwdtool
一个简单粗暴的密码管理器，存储你不太重要但是需要记住的网站、用户密码等
基于Python3.6开发，无需安装库，在终端下运行hellp可得到帮助 ---->

E:\文件\Python脚本\密码管理器>pwdtool.py -help

        -help       显示帮助列表
        -n          创建新的数据库及表
        -a          插入数据
        -u          更新数据
        -d          删除数据
        -s          查询数据
        -del        删除数据库（慎用）
        查询元素是指id,username,password这类变量
